## :information_source: How To Use

### :rocket: Backend

```bash
# Clone this repository
$ git clone https://github.com/eltonlazzarin/portfolio-page

# Go into the repository
$ cd portfolio-page

# Install dependencies
$ yarn
```

### :rocket: Frontend

```bash
# Run the app
$ yarn start

Open http://localhost:5000 in your browser
```
